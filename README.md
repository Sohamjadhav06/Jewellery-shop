# Potdar Jewellery - Modern Heritage & Timeless Craft

Welcome to the **Potdar Jewellery** website project. Established in 1954, Potdar Jewellery blends tradition with modern design to offer exquisite, high-quality jewellery.

## 🌟 Key Features

-   **Live Gold Rates**: Real-time pricing for 24K and 22K gold.
-   **Curated Collections**: Explore Gold, Diamond, Earrings, Necklaces, Rings, and Mangalsutras.
-   **Luxury Experience**: Featuring heritage craftsmanship with a modern digital storefront.
-   **Store Locator**: Find our physical locations, including our Palus store.
-   **Integrated Checkout**: A seamless shopping experience from selection to purchase.

## 🚀 Getting Started

### Prerequisites

To run this project locally, you only need a simple HTTP server. If you have Node.js installed, you can use `http-server`.

### Running Locally

1.  Open your terminal in the project directory.
2.  Run the following command:
    ```bash
    npx http-server ./ -p 8080
    ```
3.  Open your browser and navigate to: [http://localhost:8080/index.html](http://localhost:8080/index.html)

## 📂 Project Structure

-   `index.html`: The main homepage featuring hero sections, highlights, and navigation.
-   `collections.html`: Browse through our diverse jewellery categories.
-   `product.html`: Detailed product views for individual pieces.
-   `store.html`: Information about our physical stores.
-   `checkout.html`: The final step in the customer's journey.
-   `style.css`: Custom styling for a premium look and feel.
-   `script.js`: Interactive elements and dynamic features.
